## #2. 강조 Emphasis

### ✅ em

```html
<em>강조하는 내용</em>
```

### ✅ strong

```html
<strong>강조하는 내용</strong>
```

## #3. 링크 Anchor

```html
<a href="주소">링크</a>
```

### ✅ href 표기법

- 웹 url
- 페이지 내 이동
- 메일 쓰기
    
    ```html
    <a href="mailto:메일주소">메일주소</a>
    ```
    
- 전화 걸기
    
    ```html
    <a href="tel:전화번호">전화번호</a>
    ```
<br/>
    
### ✅ target

```html
<a target="_blank"></a>
```

- 새 창을 열어 url 이동
