# git 파일 올리기

1️⃣ 터미널 열기

2️⃣ `git add .`

- .은 모든 걸 add한다는 뜻

3️⃣ `git commit -m "메시지"`
4️⃣ `git push`

- 만약 여기서 오류 발생하면 다음과 같이 입력
- git push origin master
